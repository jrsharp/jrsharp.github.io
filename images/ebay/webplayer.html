This auction is for a Virgin Webplayer internet appliance with 2.5" IDE cable. This webplayer unit is in good shape and great working condition. The video memory chip responsible for the bottom half of the screen has gone bad. This problem is a common one and is actually quite tolerable in text mode. The LCD is in fine shape and the chip problem only garbles the output on the bottom half. The disk-on-chip (DOC) and DOC door are missing. This is not a big deal, really, especially if you plan to run a real operating system on it via a laptop hard drive. It does not have the modem module. Adding a USB ethernet adapter is trivial.

The webplayer comes with the base unit with screen, the wireless keyboard/trackball, power supply, and 2.5" IDE cable. I have successfully installed and used Windows 98, Windows ME, Slackware Linux 8.0 and Redhat Linux on this unit.
